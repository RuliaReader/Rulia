# @rulia/types

This package provides the type definition for developing Rulia plugins.

If you don't use TypeScript, please ignore this pacakge.

## Quick start

1. Install:

```
npm install @rulia/types --save-dev
```

2. Update your `tsconfig.json`:

```json
{
  "compilerOptions": {
    "types": [
      "@rulia/types"
    ]
  }
}
```

3. Done.
