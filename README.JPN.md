# Rulia - Windows用のモダンなマンガリーダー

新しい読書体験を提供するマンガリーダー。

Microsoft Storeから入手：

<a href="https://apps.microsoft.com/store/detail/9MVVLRZWRXX8?cid=github&launch=true&mode=mini">
  <img src="https://get.microsoft.com/images/ja-jp%20dark.svg" width="250" />
</a>

## 特徴

 - 一般的なファイル形式に対応：CBR, CBZ, ZIP, RAR, 7Z, PDF, EPUB, MOBI
 - WebPや[AVIF](https://github.com/RuliaReader/Rulia/wiki/AVIF-Support)を含む様々な画像形式に対応
 - 多様なニーズに対応する複数の閲覧モード
 - 画像のモアレ（干渉縞）を除去
 - 画像の余白を自動でトリミング
 - [超解像技術](https://github.com/RuliaReader/Rulia/wiki/Super%E2%80%90resolution-for-manga-images)による低解像度画像の高画質化
 - 暗号化された圧縮ファイルの閲覧とパスワード記憶に対応
 - タッチ操作に最適化されたモダンなユーザーインターフェース
 - 複数のプロファイル作成によるマンガコレクションの管理
 - Surface PenとDialによるページめくりに対応
 - 機能拡張のための[プラグインシステム](https://github.com/RuliaReader/Rulia/wiki/Plugin-system)
